# Data quering

To query data, call "dbo.spFindPOIs" procedure and provide search criteria in the following json format:
```json
{
   "country":"",
   "region":"",
   "city":"",
   "location":{
      "lat": null,
      "lon": null,
      "radius": null
   },
   "polygon":"POLYGON ((..))",
   "poi_category":"",
   "poi_name":""
}
```

# Procedure call
  
  ```
  EXEC [dbo].[spFindPOIs]	@json = 
  '{
   "country":"US",
   "region":"AZ",
   "city":"Phoenix",
   "location":{
      "lat":33.606365,
      "lon":-112.064371,
      "radius":null
   },
   "polygon":"POLYGON (( -112.076939164701216 33.498905963129644, -112.060361302493078 33.498776448581147, -112.059843244299074 33.477406548078477, -112.076550621055702 33.477147518981475, -112.076550621055702 33.477147518981475, -112.076939164701216 33.498905963129644 ))",
   "poi_category":"Health",
   "poi_name":"c"
}'
```
